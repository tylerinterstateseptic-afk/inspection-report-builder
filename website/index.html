Setup Guide

The app includes a built-in setup wizard that walks you through each step on first launch. This page is a detailed reference if you need it.

1

Enable 2-Step Verification on your Google Account

Go to myaccount.google.com → Security → 2-Step Verification → Turn it on. This is required to generate an App Password.

2

Generate a Gmail App Password

Go to myaccount.google.com/apppasswords. Select "Mail" as the app, give it a name like "Inspection Report Builder", and click Generate. Copy the 16-character password.

3

Enter SMTP Settings in the App

The setup wizard handles this automatically on first launch, or open Settings → Email Configuration:

SMTP Host: smtp.gmail.com
SMTP Port: 587
Email / Username: your-email@gmail.com
Password: (paste your 16-char App Password)
Other email providers: For Outlook, use smtp.office365.com port 587. For Yahoo, use smtp.mail.yahoo.com port 465. Each provider may require generating an app-specific password.
1

Install Google Drive for Desktop

Download and install Google Drive for Desktop from Google. This creates a synced folder on your computer (usually G:\My Drive on Windows).

2

Find Your Drive Folder Path

Open File Explorer. Look for "Google Drive" in the sidebar. The path is usually G:\My Drive or C:\Users\YourName\Google Drive. You can create a subfolder like G:\My Drive\Inspection Reports.

3

Enter the Path in the App

The setup wizard asks for this on first launch, or open Settings → paste the full folder path into the Google Drive Folder Path field. Every PDF you generate will automatically be copied there and synced to the cloud.

1

Create an Anthropic Account

Go to console.anthropic.com and create a free account. You'll need to add a payment method for API usage (pay-per-use, typically just cents per report).

2

Generate an API Key

In your Anthropic dashboard, go to API Keys → click Create Key. Copy the key — it starts with sk-ant-...

3

Paste the Key in the App

The setup wizard asks for this on first launch, or open Settings → paste your API key into the Claude API Key field. Then click "AI Improve" on any text field to polish your notes.

Cost: The AI feature uses Claude's API on a pay-per-use basis. Each text improvement costs roughly $0.01-0.03. A typical month of heavy use might cost $5-10. Your API key is stored locally on your computer and never shared.
1

Set Your Company Information

Open Settings → fill in the Company Branding section: company name, address, phone number, tagline, and established date. This information appears on every generated report.

2

Choose Your Fonts

Select a Report Body Font and a Company Name Font from the dropdowns. The company name font is used for the large header on your PDF reports.

3

Customize Your Disclaimers

Scroll down in Settings to the Report Disclaimers section. Edit the septic and sewer disclaimers to match your local regulations and liability needs. Use {companyName} as a placeholder — it will be automatically replaced with your company name.

4

Customize Fields & Sections

Click Edit Fields in the top bar to enter Edit Mode. You can rename field labels, edit dropdown options, add new custom fields, add entire new sections, reorder items, and hide fields you don't need.
