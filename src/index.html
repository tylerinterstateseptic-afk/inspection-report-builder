✉ Email Setup

Connect your email to send reports directly to clients with the PDF attached.

How to get a Gmail App Password:

  1. Go to myaccount.google.com → Security → enable 2-Step Verification
  2. Then go to myaccount.google.com/apppasswords
  3. Name it "Inspection Report Builder" and click Generate
  4. Copy the 16-character password and paste it above

For Outlook use smtp.office365.com port 587. For Yahoo use smtp.mail.yahoo.com port 465.

☁ Google Drive Setup

Auto-copy every generated PDF to a Google Drive folder for cloud backup.

Setup steps:

  1. Download and install Google Drive for Desktop from Google
  2. It creates a synced folder — usually G:\My Drive on Windows or /Volumes/GoogleDrive/My Drive on Mac
  3. Optionally create a subfolder like G:\My Drive\Inspection Reports
  4. Paste the full path above

Every PDF you generate will be automatically copied to this folder and synced to the cloud. If you don't use Google Drive, skip this step.

🤖 AI Writing Assistant

Connect Claude AI to turn rough field notes into polished, professional language with one click.

How to get your API key:

  1. Go to console.anthropic.com and create an account
  2. Add a payment method (pay-per-use, just cents per report)
  3. Go to API Keys → click Create Key
  4. Copy the key (starts with sk-ant-...) and paste it above

Cost: Each text improvement costs roughly $0.01–0.03. A typical month of heavy use is about $5–10. Your key is stored locally on your computer and never shared with anyone.
